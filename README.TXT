PROJECT TITLE: Website for athletes
PURPOSE OF PROJECT: The purpose of this project is to manage athletes, their teams, equipments and activities
VERSION or DATE: April 28,2025
HOW TO START THIS PROJECT: 
AUTHORS: Fritz Gerald Sarabia, Dylan Scalia, Gabriel Roberts-Palacios
USER INSTRUCTIONS:
Open the project in BlueJ or any compatible Java IDE. Compile all classes. Create instances of Teams, Coaches, Athletes, Activities, and Equipment through the IDE or by writing a simple main method for testing.
- Create a Team object and assign a Coach to it.
- Create Athlete objects and add them to the Team.
- Assign Activity and Equipment objects to each Athlete.
- Use getter methods to retrieve athlete details, team rosters, activities, and equipment lists.
